# Memos Plus

Memos Plus is a local-first capture and delivery workbench for Obsidian.

It helps you collect text, links, images, quick thoughts, tasks, and temporary material, then send them directly to the right Markdown file, project note, tag-based note, template-created file, or real Markdown heading.

Instead of becoming another inbox you need to clean later, Memos Plus turns capture into the first step of organization.

## Why It Exists

Obsidian is excellent for long-lived notes, but small incoming fragments often arrive faster than you can file them:

- A useful link you want to keep with context.
- A project idea that belongs under an existing heading.
- A task that should become an Obsidian Tasks item.
- A medical note, case detail, or reference that belongs in a topic file.
- A quick mobile thought that should not interrupt your current work.

Memos Plus gives those fragments a fast entry point and a clear destination.

## Highlights

- **Local-first Markdown storage**: memos are saved as readable Markdown files in your vault.
- **Multiple capture surfaces**: use the main workbench, quick capture modal, sidebar quick input, or mobile entry.
- **Send to real notes**: deliver content to project files, tag-filtered files, recent files, search results, or specific Markdown headings.
- **Template-based file creation**: when the target note does not exist, create it from a configured Markdown template and continue the same send flow.
- **Flexible content formats**: save as a normal note, link material, task, callout, code block, or custom format.
- **Obsidian Tasks support**: create task lines with priority, dates, recurrence, and optional detail content.
- **Review and filter views**: browse by search, tags, status, pinned/starred/archived states, dates, and task-oriented sections.
- **Mobile-friendly workflow**: optimized mobile screens keep capture and file selection stable on iPhone.
- **Private by default**: no accounts, telemetry, ads, or required cloud service.

## What Makes It Different

Many memo tools stop at capture. Memos Plus focuses on the whole path:

1. Capture something quickly.
2. Decide what kind of content it is.
3. Choose where it belongs.
4. Insert it under the correct file, heading, template, or task format.
5. Review unfinished material later from focused organization views.

That makes it useful not only as a memo list, but as a practical intake layer for an existing Obsidian knowledge base.

## Common Workflows

- Collect a link and send it to a topic note's reference section.
- Capture a project idea and insert it under a development backlog heading.
- Turn an input into a task with due date and priority.
- Create a new note from a template when search finds no matching target.
- Use the mobile lightweight home to capture first and organize later.
- Review unorganized, linked, image-based, or unfinished-task items from the workbench.

## Storage

By default, Memos Plus stores yearly memo files under a configurable folder such as:

```text
我的资源/Memos/2026.md
```

The files remain ordinary Markdown and can be opened, searched, synced, backed up, or edited outside the plugin.

## Privacy

Memos Plus does not require an account and does not include telemetry or advertising. It reads and writes the Markdown files and attachments you configure inside your Obsidian vault.
